## Core Functionality

### Main Flow (viper_scan.py)

```python
from modules.nmap_integration import scan_web
from modules.burp_exporter import export_burp_issues
from modules.nikto_wrapper import run_nikto

def main(target_url: str):
    print(f"[+] Starting scan for {target_url}")
    nmap_results = scan_web(target_url)
    print(f"[+] Nmap Results: {nmap_results}")

    nikto_results = run_nikto(target_url)
    print(f"[+] Nikto Results: {nikto_results}")

    export_burp_issues(target_url, nikto_results, nmap_results)
    print("[+] HTML report generated at reports/scan_report.html")
```

**Explanation:**

- `scan_web` runs an Nmap scan for open endpoints and services.
- `run_nikto` runs basic vulnerability checks using Nikto.
- `export_burp_issues` consolidates findings (from both tools) into a readable HTML report.

### HTML Report Generation (modules/burp_exporter.py)

```python
from jinja2 import Template

HTML_TEMPLATE = """

<html>
  <head>
    <title>ViperScan Report for {{ target }}</title>
  </head>
  <body>
    <h1>Scan Report</h1>
    <h2>Nmap Findings</h2>
    <pre>{{ nmap }}</pre>
    <h2>Nikto Findings</h2>
    <pre>{{ nikto }}</pre>
  </body>
</html>
"""

def export_burp_issues(target: str, nikto_result: str, nmap_result: str):
    rendered = Template(HTML_TEMPLATE).render(
        target=target, nikto=nikto_result, nmap=nmap_result
    )
    with open('reports/scan_report.html', 'w') as f:
        f.write(rendered)
```

**Explanation:** Uses Jinja2 to format data into a human-friendly HTML file, improving readability and shareability.
