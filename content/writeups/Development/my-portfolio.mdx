# ⚡ GlitchViper.dev – Cybersecurity Portfolio

GlitchViper.dev is my personal cybersecurity-themed portfolio — a modern, animated platform built to showcase my technical expertise, projects, and writeups. It’s not just a portfolio; it’s a living demo of my skills in frontend development, automation, and cybersecurity.

## 🚀 Key Features

- 🖥️ **Built with Next.js & TypeScript**  
  Leveraging the power of Next.js 15+ for blazing-fast rendering and routing, combined with TypeScript for scalability and reliability in development.

- 🎨 **Tailwind CSS + Aceternity UI**  
  A responsive design powered by Tailwind and enhanced with Aceternity UI, ensuring a sleek, modern interface.

- 🌗 **Light & Dark Themes**  
  Full theme toggle support to match user preferences and maintain accessibility.

- 🎬 **Framer Motion Animations**  
  Smooth glitch effects, page transitions, and micro-animations create an immersive cybersecurity vibe.

- 🔍 **SEO-Optimized**  
  Structured metadata, semantic HTML, and optimized loading for high search visibility.

- 📑 **MDX-Powered Writeups**  
  Vulnerability reports (like Reflected XSS in Login Page) and bug bounty notes are written in Markdown/MDX, making it simple to maintain and publish professional documentation.

## 🛠️ Tech Stack

- ⚡ **Next.js 15+** – Frontend framework
- 🟦 **TypeScript** – Strongly typed JavaScript
- 🎨 **Tailwind CSS** – Utility-first styling
- 🧩 **Aceternity UI** – Clean UI components
- 🎬 **Framer Motion** – Animations & transitions
- 🔍 **SEO Tools** – Metadata & optimization

## 🔐 Cybersecurity Showcase

- 🐍 **ViperScan** – A Python-based web vulnerability scanner with Nmap integration and HTML reporting. (Future: AI + automated scanning)
- 🛡️ **Network Engineering Skills** – Configuring firewalls, VPNs, routing, Active Directory, and switches.
- 💻 **Programming & Scripting** – Expertise in Python, C++, Bash, and PowerShell for automation and exploitation.
- 🎯 **Bug Bounty & CTFs** – Active on Hack The Box and HackerOne, focusing on vulnerability discovery and security research.
