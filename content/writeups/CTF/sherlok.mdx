# 🐍 ViperScan – Python Web Vulnerability Scanner

ViperScan is a **custom-built web vulnerability scanner** written in Python.  
It integrates **Nmap** and **Nikto** to detect potential weaknesses in web applications, generating clean and easy-to-read **HTML reports**.

---

## 🚀 Features

- 🔍 **Port Scanning** using Nmap for service detection  
- 🌐 **Web Vulnerability Scanning** with Nikto  
- 📊 **HTML Reports** for structured vulnerability documentation  
- ⚡ Automation-ready architecture for bug bounty and pentesting workflows  

---

## 📂 Tech Stack

- **Python** – Core scripting  
- **Nmap** – Network mapping and port scanning  
- **Nikto** – Web vulnerability scanning
